# 施工现场管理系统

一套面向建设工程现场的台账管理系统，取代传统 Excel 多版本管理，实现数据实时更新、随地查阅。

## 项目背景

- **痛点**：多个版本的 Excel 台账、审批流程繁琐、数据分散难以追溯
- **目标**：台账管理模式，数据提交即生效，无需等待审批流程
- **适用场景**：房建、市政、公路、铁路等各类建设工程项目

## 核心特性

### 1. 台账管理模式
- 数据提交后直接进入台账，无需等待审批
- 每次操作带时间戳，确保数据可追溯
- 支持批量导出 Excel，对标传统台账格式

### 2. 智能数据采集
- **OCR识别**：发票、垫付截图自动识别
- **AI隐患识别**：现场拍照，自动识别安全隐患
- **天气API对接**：自动获取当日天气（和风天气）

### 3. 施工日志自动化
- 工程部填报当日进度后，系统自动汇总：
  - 人员投入（数量、工种、班组）
  - 机械设备（设备名称、规格、数量）
  - 材料消耗（物料名称、用量、使用部位）
- 自动生成施工日志模板，一键打印备查

### 4. 模板定制化
- 日志模板可视化编辑
- 不同项目可选择不同模板
- 字段数据来源可配置（自动获取/手动填写）

### 5. 多项目管理
- 支持同时管理多个项目
- 一个员工可兼任多个项目（主项目 + 兼项目）
- 项目切换便捷，数据隔离清晰
- 跨项目数据汇总报表

### 6. 权限体系
- 基于"用户-角色-项目"三级权限
- 支持全局角色（综合办）和项目级角色（各项目专属）
- 权限精细到模块级别

### 7. MT Photos 照片集成
- 施工现场管理人员通过 MT Photos App 上传的照片
- 项目管理系统可直接获取当日照片，关联到施工日志
- 每日凌晨自动同步前一天的照片
- 照片按拍摄时间自动归档到对应项目的当日施工日志

### 8. AI 智能功能
- **OCR识别**：发票、垫付截图自动识别（本地处理，保护隐私）
- **AI隐患识别**：现场拍照，自动识别安全隐患
- **天气API**：自动获取当日天气（和风天气）
- **AI日报总结**：施工日志内容润色和总结（可选）
